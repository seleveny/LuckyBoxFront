# 运气棒棒微信小程序

## 功能介绍

本小程序包含以下功能：

1. **个人中心** - 显示用户微信信息
2. **微信授权** - 首次使用时需要授权登录
3. **信息同步** - 授权后将用户信息发送到后端接口

## 配置说明

1. **后端接口配置**：在 `app.js` 文件中，需要将 `sendUserInfoToBackend` 函数中的接口地址替换为真实的后端接口地址。

2. **OpenID 获取**：实际项目中，需要通过微信登录接口获取用户的 OpenID，并存储到本地存储中，以便在发送用户信息时一起提交。

## 使用方法

1. 在微信开发者工具中导入本项目
2. 点击「编译」按钮运行小程序
3. 首次进入时，会提示授权登录
4. 授权后，个人中心页面会显示用户信息
5. 用户信息会自动发送到后端接口

## 注意事项

1. 本项目使用了微信小程序的 `getUserInfo` 接口，需要用户主动授权
2. 实际项目中，需要在后端服务器中处理用户信息的存储和验证
3. 为了安全起见，建议在发送用户信息时使用 HTTPS 协议

## 开发环境

- 微信开发者工具
- 微信小程序基础库 2.0.0 及以上
